Ce projet consiste  la conception et la réalisation d'un Curriculum Vitae (CV) interactif sous forme de page web. 



##  Technologies Utilisées

HTML : Structuration sémantique de la page.
CSS : Mise en forme, transitions, animations et media queries responsives.
JavaScript  : Interactivité complète .
FontAwesome  : Bibliothèque d'icônes .
Google Fonts: fonts des textes personnalisées.



## Instructions pour Ouvrir la Page


Pour le lancer :
1. éléchargez ou cliquez sur le dossier du projet.
2. Ouvrez le fichier index.html dans n'importe quel navigateur web.
    Soit par double-clic sur le fichier.
    Soit par glisser-déposer  du fichier dans la fenêtre du navigateur.
3. Testez l'interactivité :
    Cliquez sur le bouton d'engrenage à droite pour changer la couleur principale.
    Cliquez sur le bouton de lune/soleil pour basculer le mode sombre.
    Remplissez le formulaire de contact pour tester la validation.
    Réduisez la largeur de votre écran pour tester la version mobile.
